# RIFTBOUND: Fantasy Action-Platformer Roguelike
Guide the player through everchanging runs across various diverse worlds.

# Supported Operating System(s):
	- Windows 11

# Controls:
	- W / Up Arrow / Space: Jump
	- A / Left Arrow: Move Left
	- D / Right Arrow: Move Right
	- S / Down Arrow: Fast Fall
	- Shift: Sprint/Run
	- Left Click / Z: Attack
