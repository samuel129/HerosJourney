# Hero's Journey: Fantasy Action-Platformer Roguelike
A game about a Hero on a Journey.

# The Game Loop: 
2d platformer where you explore different levels of platforms, encountering various enemies of varying strengths as you progress to the end where you encounter mini or main bosses. Lose items, perks, stats upon death. 

# The data in our Game State: 
In run: General stats of player(hp, perks, abilities) structures, enemies

# What would go into a save file & what would be procedurally generated: 
Procedurally generated: Entities, levels/rooms
Save file: player stats, enemy types, stats associated, & runtime

# How you would turn an unfun or winnable situation into a fun one again (or why you wouldn't):
We wouldn't, we want to make it difficult for a player, they should not be able to beat the game on their first try. We want the game to be hard & punishing if you lose, as the player will get better at the game with time.
